Many of the most influential and big businesses in the world today are tech businesses.Moore's Law is a trend that states: Every year or two, the capacities of computers have doubled inexpensively.Computers and the Internet are being used in industry, research, health care, social good, communication, entertainment and more. In this section we would just be giving you some useful facts, terms, vocabulary and abbrievations related to the web (the internet):

  • A bit is the smallest data item in a computer; it can have the value of 0 or 1.
  • The internet was made posibble by a gloabal convergence of computing and communication technologies.
  • The Advanced Research Projects Agency Network (ARPANET) was the basis for the internet.
  • One of the primary goals for ARPANET was to allow multiple users to send and receive information simultaneously over the same communications paths

  • The set of protocol(rules) for the internet and the APRANET is called TCP/IP.
  • HTTPS means Hypertext Transfer Protocol Secure
  • SSL means Secure Sockets Layer
  • TLS means Transport Layer Security
  • URI means Uniform Resource Identifier
  • The two most common HTTP request types (also known as request methods) are get and post.
  • A get request retrieves information from a server.
  • A post request typically posts (or sends) data to a server.
  • Browsers often cache (save on disk) recently viewed web pages for quick reloading.

CSS background properties are used to determine the background of an element, such as the general color of the page or the baground of a div. Below are some common CSS background properties :
  • background-color : Its specifies the background color of an element.
  • background-image : It specifies an image to use as the background of element.
  • background-repeat :It specifies how you want and image background to be repeated, usually repeat-x, repeat-y or n-repeat
  • background-attachment : Specifies whether the background image should scroll or be fixed
  • background-position: Specifies the starting point of the background image

When using the shorthand property the order of the property values is:
  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
    • It does not matter if one of the property values is missing, as long as the other ones are in this order.

    Color names and hexadecimal codes may be used as the color property value. You can also find a complete list of HTML standard and extended colors at www.w3.org/TR/css3-color/ or https://www.w3schools.com/colors/colors_picker.asp
