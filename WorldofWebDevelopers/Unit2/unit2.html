Control Statements in JavaScript

All scripts can be written in term of three control Statements:

  1. sequence
  2. selection
  3. repition

Sequencce Statement

A sequence is a set of instructions executed one after the other

Selection Structures

A selection statement is a conditional statement. Javascript has 3 selection structures:

  • if statement (single selection)
  • if..else statement (double selection)
  • switch statement (multiple selection)

Iteration/Repition Statements

This allows us to repeat a certain statement for a certain number of times whilst the condition is true. Javascript has 4 repition statements:

  • while
  • do...while
  • for
  • for...in

Keywords

Functions in JavaScript

In any program you are writing to maintain it , it is best written in small, simple pieces this helps you solve each chunk of the algorithm easily. This is where functions are useful in programming, you can define functions that perform specific task and use them at many points in a script.

Functions in javaScript are defined like this:

        

        function function-name(parameter-list)

        {

              declarations and statements
        }

They are called like this:

        

        function-name(parameter-list);

Methods are called the same way as functions but require the name of the object to which the method belongs and a dot preceding the method name.

There are several reasons for modularizing a program with functions some of which are:

  • Divide-and-conquer aproach makes program development more manageable.
  • Software reusability
  • Avoid repeating code in a programme

JavaScript also comes with a lot of Global functions. Here are some of the popular ones.

  • parseInt(): Parses an string and returns an interger
  • parseFloat(): Parses an string and returns a floating point number
  • Number():Converts an object's value to a number
  • String(): Converts an object's value to a string
  • isFinite(): Determines whether a value is finite, legal number
  • isNAN(): Determines whether a value is an illegal number
  • decodeURI():Decodes a URI
  • decodeURIComponent():Decodes a URI component
  • encodeURI():Encodes a URI
  • decodeURIComponent ():Encodes a URI component

Arrays, Objects and Document Object Model(DOM) in JavaScript

Arrays

Arrays are data structures consisting of related items. They can also be defined as a group of memory locations of the same name and most times the same type. Arrays in javaScript are dynamic entities as they can change size after they are created. The first element of a JavaScript array usually takes index zero, the second takes index one and so forth. In JavaScript we use the new operator to create an array and to specify the number of elements in an array. JavaScript's for....in Repitition statement enable a script to perform a task for each element in an array


Did you know that Javascript does not check the number of arguemnets or types of arguements passed to a function ? It's possible to pass any number of values to a function

The array object in Javascript has a built-in sort method : Sorts the array either in alphabetical order or numeric order

The array object in Javascript has a built-in indexOf method : Searches for the first occurence of the specified key value

The array object in Javascript has a built-in lastIndexOf method : Searching for the last occurence of the specifies key value

Arrays in JavaScript can also be two dimensional, to call an element in a two dimensional array you usaully have the specify the two indices. By convention, the first identifies the element's row and the second identifies the element's column.
